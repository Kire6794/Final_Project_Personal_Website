Searchable notes (15+ concepts)

Use the search bar + tags to filter. This page is intentionally “developer handy.”

All DOM Arrays Async Syntax

1) document.querySelector()

Selects the first matching element.

const btn = document.querySelector("#saveBtn");

2) document.querySelectorAll()

Selects all matching elements (NodeList).

const items = document.querySelectorAll(".card");

3) addEventListener()

Respond to clicks, input, submit, keypress, etc.

btn.addEventListener("click", () => alert("Hi!"));

4) event.preventDefault()

Stops default browser behavior (great for forms).

form.addEventListener("submit", (e) => { e.preventDefault(); });

5) Template literals

Use backticks to embed variables.

const msg = `Hello ${name}!`;

6) Destructuring

Pull values from objects/arrays quickly.

const { id, title } = post; const [a,b] = nums;

7) Array.map()

Transforms items into a new array.

const names = users.map(u => u.name);

8) Array.filter()

Keeps items that match a condition.

const adults = users.filter(u => u.age >= 18);

9) Array.find()

Returns the first match (or undefined).

const user = users.find(u => u.id === 7);

10) Array.reduce()

Combines array into one value (sum, object map, etc.).

const total = prices.reduce((s,p) => s + p, 0);

11) localStorage

Persist small data in the browser.

localStorage.setItem("theme","dark"); const t = localStorage.getItem("theme");

12) JSON.parse / JSON.stringify

Convert between objects and strings.

const s = JSON.stringify(obj); const o = JSON.parse(s);

13) fetch()

HTTP requests (usually returns JSON).

const res = await fetch("/api/data"); const data = await res.json();

14) try/catch

Handle errors safely.

try { risky(); } catch (err) { console.error(err); }

15) async/await

Write asynchronous code that reads like synchronous.

async function load(){ const r = await fetch(url); return r.json(); }

16) Set / Map

Unique values (Set) and key/value pairs (Map).

const s = new Set([1,1,2]); const m = new Map([["a",1]]);
